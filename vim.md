#### vim
-------------------------------
```

vi xxx 进入文件
v：进入可视化状态  在可视化下面的操作
p：黏贴;
y：拷贝选择的内容到剪贴板。
c:：剪贴选择的内容到剪贴板并且进入插入模式
d：删除；
```
