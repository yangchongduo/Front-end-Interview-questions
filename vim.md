#### vim
-------------------------------
```

vi xxx 进入文件
v：进入可视化状态  在可视化下面的操作
p：黏贴;
y：拷贝选择的内容到剪贴板。
c:：剪贴选择的内容到剪贴板并且进入插入模式
d：删除；
```

#### 如何设置有行号
  cp /usr/share/vim/vimrc ~/.vimrc

  先复制一份vim配置模板到个人目录下

  注：redhat 改成 cp /etc/vimrc ~/.vimrc

步骤2：

  vi ~/.vimrc

  进入insert模式，在最后加二行

  syntax on

  set nu!

保存收工。
